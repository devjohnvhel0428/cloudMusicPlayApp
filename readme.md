# OMP - OneDrive Media Player

Open your browser and **[Now Play](https://nini22p.github.io/omp/)**

## Development

Add `.env.development`

```env
VITE_CLIENT_ID=<clientId>
VITE_REDIRECTURI=http://localhost:5173/
```

Run `npm i && npm run dev`
